DOM Play

This is a webpage that displays a snippet of a Naruto episode script. It will highlight all of each person's lines when you click on a line of the script.


A single web page wired with events: a single page application using the DOM. The page:

  • With a single click, highlight(bold or otherwise) all lines for a specific actor in a play
  • Un-highlight (normal or otherwise) all lines from all other actors in the play
  • The function(s) created to do the work must accommodate any number of actors, and any number of additional lines aded to the page
  • The page does the work without page re-loads (ie. done via JS and the DOM)
  • Page does not use a third party API
